Common Problems & Solutions

Step-by-step guides to fix issues yourself

Phone Not Charging / Battery Draining Fast

Quick Fixes:

  1. Clean charging port: Use a toothpick to remove lint/dust carefully
  2. Try different cable/charger: Test with original or certified charger
  3. Restart phone: Force restart to clear software glitches
  4. Check for background apps: Close apps draining battery (Settings > Battery)
  5. Update software: Install latest OS updates
Still not working? Battery may need replacement (₹500-₹2000 depending on model).

Phone Hanging / Freezing / Slow Performance

Step-by-Step Solution:

  1. Clear cache: Settings > Storage > Cached Data > Clear
  2. Uninstall unused apps: Remove apps you don't use
  3. Free up storage: Keep at least 15% storage free
  4. Disable animations: Developer Options > Reduce animations
  5. Factory reset (last resort): Backup data first, then Settings > Reset
Prevention tip: Avoid installing too many apps. Use lite versions when possible.

Laptop Running Slow / Hanging

Quick Fixes:

  1. Check RAM usage: Task Manager (Ctrl+Shift+Esc) > Performance
  2. Close startup programs: Task Manager > Startup > Disable unwanted
  3. Run Disk Cleanup: Search "Disk Cleanup" > Clean system files
  4. Update drivers: Device Manager > Update all drivers
  5. Scan for malware: Run Windows Defender or Malwarebytes
  6. Consider SSD upgrade: Biggest speed boost (₹2000-₹5000)
Pro Tip: 8GB RAM + SSD = Instant 5x speed boost on old laptops!

Windows 10/11 Not Booting / Blue Screen Error

Emergency Steps:

  1. Safe Mode: Restart > F8 repeatedly > Safe Mode
  2. Startup Repair: Boot from USB > Repair your computer
  3. System Restore: Restore to previous working state
  4. Check hardware: Remove external devices and try again
  5. Last resort - Reinstall Windows: Backup data via live USB first
⚠️ Critical: Don't panic! Most issues are fixable. Contact us before formatting.

Microsoft Office Not Activating / Trial Expired

Activation Solutions:

  1. Use genuine key: Enter 25-digit product key (File > Account)
  2. Online activation: Connect to internet and activate
  3. Phone activation: Call Microsoft if internet fails
  4. Remove previous activation: Uninstall completely, clean registry, reinstall
Need a license? We provide genuine Office 2021 licenses at ₹4,899 (lifetime).

WiFi Connected But No Internet Access

Troubleshooting Steps:

  1. Restart router: Unplug for 30 seconds, plug back
  2. Forget & reconnect WiFi: Forget network, re-enter password
  3. Flush DNS: CMD > Type "ipconfig /flushdns"
  4. Reset network settings: Settings > Network Reset
  5. Update WiFi driver: Device Manager > Network Adapters
Check: Is the issue on all devices? Contact ISP if yes.

Recover Deleted Photos / Files / WhatsApp Chats

Recovery Methods:

  1. Check Recycle Bin / Recently Deleted: First place to look
  2. Google Photos / Cloud backup: Check auto-backup
  3. Use Recuva (PC): Free software for deleted file recovery
  4. WhatsApp local backup: Internal Storage > WhatsApp > Databases
  5. Professional recovery: For formatted drives / physical damage
⚠️ Important: Stop using device immediately to prevent overwriting!
